# Structured Brainstorming

Generate diverse, actionable ideas using proven creativity frameworks.

## Process

### Phase 1: Frame the Problem (2 min)
- Restate the problem/opportunity in one sentence
- Identify constraints (real ones, not assumed ones)
- Ask: "What would success look like?"

### Phase 2: Diverge - Generate Ideas (bulk of time)
Use at least 3 of these frameworks:

**Bisociation**: Combine the problem domain with an unrelated domain
- "What if [your problem] worked like [restaurants/airports/video games/hospitals]?"

**Constraint Manipulation**:
- Remove a constraint: "What if budget/time/tech wasn't a limit?"
- Add a constraint: "What if it had to work offline/be voice-only/fit on one screen?"

**Role Storming**:
- "How would a game designer solve this? A teacher? A 5-year-old? An adversary?"

**Reverse Brainstorm**:
- "How would we make this problem WORSE?" then flip each answer

**10x Thinking**:
- "What if we needed to serve 10x the users? Cost 10x less? Ship in 1/10th the time?"

Target: 10-15 raw ideas. No filtering yet.

### Phase 3: Converge - Evaluate
1. **Cluster** related ideas into 3-5 themes
2. **Score** each cluster: Feasibility (1-5) x Impact (1-5)
3. **Identify** top 3 ideas with best score
4. **Develop** each into a concrete next step

### Phase 4: Action
For each top idea, define:
- What it is (one sentence)
- First concrete step to try it
- How you'd know if it's working
- Biggest risk

### Output Options
- **Quick mode**: Just the top 3 ideas with next steps
- **Full mode**: All ideas clustered, scored, and developed
- **Save mode**: Write to BRAINSTORM.md for later reference
