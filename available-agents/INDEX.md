# Available Agents Library

Specialized agents ready to install into any project. Use `/skill-store` to browse and install.

| Agent | Model | Description |
|-------|-------|-------------|
| `brainstormer` | sonnet | Creative ideation specialist with divergent/convergent thinking frameworks |
| `brand-voice-expert` | sonnet | Brand voice, tone profiles, content pillars, audience personas, anti-AI-slop |
| `creative-writer` | opus | Blog posts, articles, copywriting, social media. Anti-AI checklist built in |
| `fullstack-dev` | sonnet | JS/TS, React, Next.js, Node, databases, APIs. Type-safe end-to-end |
| `product-designer` | opus | PRDs, user stories, acceptance criteria, roadmaps. Grill-me + plan-and-spec |
| `seo-expert` | sonnet | On-page SEO, content optimization, technical SEO, AI search optimization |
| `web-designer` | opus | HTML/CSS/JS, React, accessibility (WCAG 2.1 AA), responsive, performance |
